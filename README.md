Guida per l'utilizzo del programma

1)Come creare un file dotnet<br />
  aprire il terminale e digitare:<br />
```
dotnet new console
```
2)Come aggiungere la libreria "SQLITE"<br />
  digitare nel terminale:
```
dotnet add package sqlite-net-pcl
```
